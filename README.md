# 🏨 Hotel Management System

## 📌 Overview

The **Hotel Management System** is a Java-based desktop application designed to manage hotel operations such as room booking, check-in, and check-out efficiently. It provides a simple and interactive interface to handle room availability and customer flow.

---

## 🚀 Features

* 🛏️ Book rooms based on availability
* 🔑 Check-in and Check-out system
* 📋 View available and occupied rooms
* 🚫 Prevent double booking of rooms
* 🎯 Dynamic room selection 
* 🎨 User-friendly interface using JavaFX

---

## 🛠️ Tech Stack

* **Language:** Java
* **GUI:** JavaFX
* **IDE:** IntelliJ IDEA
* **Version Control:** Git & GitHub

## ▶️ How to Run the Project

1. Clone the repository:

   ```
   git clone https://github.com/GarimaDixit2502/hotelmanagement.git
   ```
2. Open the project in IntelliJ IDEA
3. Ensure JavaFX is properly configured
4. Run `Main.java`
5. The application window will launch

---

## 💡 Key Functionalities

* Displays only **available rooms** during booking
* Displays only **occupied rooms** during checkout
* Maintains real-time room status
* Ensures smooth and error-free booking flow
